# Sorting Algorithm Visualizer

This is a sorting algorithm visualizer made with RayLib and C

## Supported algorithms
- [x] Bubble Sort
- [x] Selection Sort
- [x] Insertion Sort
- [x] Merge Sort
- [x] Quick Sort
- [x] Heap Sort
- [x] Cycle Sort
